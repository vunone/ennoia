# Code of Conduct

Ennoia adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
as its code of conduct.

## Summary of expected behavior

- Be respectful, inclusive, and collaborative.
- Give and receive constructive feedback on technical work.
- Accept responsibility and apologize when mistakes happen.
- Focus on what is best not only for ourselves as individuals but for the
  broader community of Ennoia users and contributors.

## Enforcement

Report unacceptable behavior to the maintainer team at
`conduct@ennoia.dev`. All complaints are reviewed and investigated
promptly. Maintainers are obligated to respect the privacy and security
of the reporter of any incident.

The full text of the Contributor Covenant v2.1 governs this project —
consult the linked document for the complete set of standards and
enforcement guidelines.
